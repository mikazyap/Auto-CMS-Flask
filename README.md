# Auto-CMS
Made with Python 3.7 and Flask

## How to Run
Prerequisites:
- `Python 3.7`
- Git
- `Pipenv`

Install `Pipenv`
```sh
$ pip install pipenv
# on Ubuntu
$ pip3 install pipenv
```

Clone repository
```sh
$ git clone https://github.com/mikazyap/Auto-CMS-Flask.git
```

Install dependencies
```sh
$ cd auto-cms-flask
$ pipenv install
# on Windows
$ python -m pipenv install
```

Run server
```sh
$ pipenv run flask run
# on Windows
$ python -m pipenv run flask run
```
